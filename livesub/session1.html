Live Code Session 1

This is the first live code session, where we recreate a webpage on the fly. We will demonstrate the following skills.

  • The basic tags of any webpage, including the DOCTYPE, html, title, and body
  • A heading to indicate a topic at the top of the page
  • Paragraphs to hold bits of text
  • Unordered lists to create a collection of bulleted items
    1. Lists can be nested
    2. This nested list begins with D
    3. The style changed the type to uppercase alpha
  • Ordered list to create a sequence of items
  • Anchor tags to link to other webpages
